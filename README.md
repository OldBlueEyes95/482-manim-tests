# Manim Calculus & Linear Algebra Animations 🎥📐

This repository contains a collection of mathematical animations created using [Manim](https://www.manim.community/). These visualizations are designed to help students and educators better understand key concepts in calculus and linear algebra.

## 📂 Animations Included

### 1. **Derivative Definition Animation**
A dynamic illustration of the limit definition of the derivative using secant and tangent lines. Includes:
- Labelled points `(x, f(x))` and `(x+h, f(x+h))`
- Real-time secant slope calculation
- Visual transition as `h → 0`

### 2. **Related Rates: Cone Draining Problem**
Visualizes a classic related rates problem involving a cone with decreasing water level. Features:
- Moving water level
- Dynamic volume, height, and radius relations
- Algebraic simplification steps animated

### 3. **Matrix Multiplication Tool**
An intuitive breakdown of matrix multiplication:
- Color-coded entries
- Dot product visualization
- Step-by-step computation

### 4. **Cylindrical Shells: Volume of Solids of Revolution**
Shows how solids of revolution are formed using the cylindrical shells method:
- Rotating region about the y-axis
- Accumulated volume with animated shell layers
- Shell height, radius, and circumference visual cues
